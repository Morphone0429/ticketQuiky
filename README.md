# 泡泡玛特 AutoJS 脚本

## 项目简介

- 基于 AutoJS/Hamibot 的泡泡玛特自动化脚本，支持 UI 弹窗配置、刷新与支付等操作。
- 提供 Hamibot 版本（规避微信无障碍混淆）与 OCR 版本（基于截图识别，适配坐标）。

## 使用步骤

1. 手机安装 AutoJS6 与 Hamibot。因微信新版混淆无障碍页面，Hamibot 改包名以白名单形式绕过。
2. 按 Hamibot 官网指南完成手机配置与授权。
3. 将脚本部署到设备：`泡泡玛特(高级).js`（UI 弹窗入口）与 `ppmt_break.js`。需具备基础 JS 能力进行配置。
4. 其他 OCR 版脚本无需 Hamibot，但需按设备屏幕分辨率调整坐标 point。

## 文件说明

- `泡泡玛特(高级).js`：主入口，提供 UI 弹窗配置与启动。
- `ppmt_break.js`：破盾相关逻辑脚本。
- 其他 OCR 脚本：基于截图识别，适配特定分辨率坐标。

## 注意事项

- 微信页面混淆可能导致无障碍控件定位异常，需使用 Hamibot 改包名方案。
- 预售场景需异步点击最后一期按钮，确保刷新后仍锁定目标期数。
- 网络卡顿时若一直停留在 loading，脚本会停止；需手动确认页面状态后重启。
- OCR 版本需根据设备分辨率更新坐标点，避免点击偏移。
