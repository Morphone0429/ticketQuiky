# 泡泡玛特 ppmt autojs 脚本

### 实现方式

1. 原始实现方式 借用了 hamibot(该软件包已重命名了无障碍包名绕开了微信的页面混淆,可以理解为改包名为白名单包名)
2. orc 截图识别实现

####

1. 倒计时刷新 ok
2. 原地无感刷新 ok
3. 控制请求速度 ok
4. 破盾暴力 (限制次数) half
5. 随时启动 兼容停止开始 ok
6. popmart loading
7. UI ok
8. 自动支付 ok
9. 预售 原地刷新需要异步重新点击最后一期按钮 ok
10. 原地刷新/破盾模式 加入缓存 取消
11. UI 打印打印信息 脚本通信
12. 原地刷新和破盾 支持实时
13. 群控
14. 额外字段 适配 A/B 组 颜色组 预售日期 ok
15. 网速卡顿时 确定页面 loading 一直存在找不到确定按钮 脚本会停止
16. 倒计时非法闯入

#### 使用说明

1. 手机安装 autojs6,Hamibot, 微信新版混淆了无障碍服务的页面,需要借助 hamibot
2. 根据 hamibot 官网介绍,设置手机配置
3. 泡泡玛特(高级).js,ppmt_break.js 两份文件就可以启动脚本,泡泡玛特(高级).js 即 UI 弹窗,需要你有基础的 js 能力
4. 其他文件采用的是 orc 能力实现的脚本 所以不需要 hamibot,但是需要手机适配屏幕的 point
